# Artify-social-networking-web-application
A basic Full stack Social Networking Application built with of HTML, CSS, and Javascript, Node.js, Express, and MongoDB featuring user authentication and follow systems. The web application demonstrates a basic follow, login, and post functionalities.  

## Project Notes
This project was developed during my second year as a learning-focused full-stack application.
The primary goal was to understand backend logic, database interaction, and authentication rather than UI design.
